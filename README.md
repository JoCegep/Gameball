# Énoncé
Vous êtes soudainement designer web, une des tâches connexes d’un programmeur.
Un client vous approche avec une idée et certains critères. Vous devez analyser (lire entre les lignes) 
son offre et décrire en terme plus technique ce qui sera nécessaire comme fonctionnalités et technologies pour réaliser le projet.
# Demandes des clients
Notre compagnie s’appelle « Meilleur Inc ». Nous sommes une petite équipe avec pleins d’idées. Ce que nous de vous, 
c’est la création d’une plateforme de jeux HTML. Un site où vous pourrez choisir entre plusieurs jeux. 
Nous voulons un look très épuré et moderne. Le site doit comprendre des tableaux des meneurs, une certaine forme de menu 
où l’on peut naviguer et sélectionner les jeux à jouer. Peut-être une vitrine pour le « jeu de la semaine ». Ça doit être très réactif et fluide.
# Analyser les besoins
Vous devez maintenant décider des fonctionnalités que le projet implique et décrire brièvement 
comment vous allez les intégrer dans le site web et quelles technologies seront utilisées. Vous pouvez même décrire 
des idées de styles visuels, de couleurs ou d’images qui vous voudriez utiliser.
